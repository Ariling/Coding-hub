- DFS함수

  - for문을 돌리면서 조건에 해당하는 경우 재귀를 돌리고 재귀 뒤에 check함수를 원래대로 돌려놓는다.
- 알파벳 바꾸기 가능 여부 함수
  - 현재 단어와 비교하려는 단어를 for문을 돌려 몇 글자만큼 다른지 체크한다.
  - 1이면 True를 반환하고 아니면 False를 반환한다.
